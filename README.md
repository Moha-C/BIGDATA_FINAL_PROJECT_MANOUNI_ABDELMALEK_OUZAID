## How to run
1. Use a base Anaconda kernel or a standard Jupyter kernel.
2. Create a temporary code cell and run:
pip install -r requirements.txt
Then delete this cell once dependencies are installed.
3. Make sure all dependencies are correctly installed.
4. Click **Run All** in the notebook.
